ü§ñ Method 1: AI-Generated Quizzes

The fastest way to create comprehensive quizzes with high-quality questions.

Step-by-Step Instructions

  1. Click "Add New Quiz" From the Admin Panel, click the green "+ Add New Quiz" button at the top of the quiz list.
  2. Enter Book Information Fill in the AI generation form:
    • Book Title: Required (e.g., "Charlotte's Web")
    • Author: Optional but recommended (e.g., "E.B. White")
  3. Choose Quiz Settings
    • Number of Questions: Select 5, 8, 10, 12, or 15 questions
    • Difficulty Level:
      Easy (Ages 8-10) Basic comprehension
      Medium (Ages 10-12) Standard analysis
      Hard (Ages 12+) Advanced critical thinking
  4. Add Additional Context (Optional) Provide specific instructions to guide the AI:
    Example: "Focus on character development and the friendship between Wilbur and Charlotte"

    Example: "Include questions about chapters 5-10 only"

    Example: "Emphasize themes of loyalty and sacrifice"
  5. Click "Generate Quiz" The AI will create your quiz in 15-30 seconds. You'll see a loading indicator while it works.
  6. Review Generated Questions After generation completes, you'll automatically be taken to the manual editor where you can:
    • Review all AI-generated questions
    • Edit question text or answer options
    • Change the correct answer
    • Add new questions
    • Delete unwanted questions
  7. Save Your Quiz When satisfied, click the "Save Quiz" button at the bottom.
‚ö†Ô∏è Important: AI-generated quizzes are loaded into the editor for review. Always check the questions before saving to ensure they're appropriate for your students.

‚úèÔ∏è Method 2: Manual Quiz Creation

Create quizzes from scratch with complete control over every question.

Step-by-Step Instructions

  1. Click "Add New Quiz" From the Admin Panel, click "+ Add New Quiz" button.
  2. Switch to Manual Mode Click the "‚úèÔ∏è Manual Entry Instead" button (or if you're already in AI mode, you can switch anytime).
  3. Enter Quiz Details
    • Book Title: Required (this becomes the quiz title)
    • Description: Optional subtitle or description
    Title: "The Chronicles of Narnia: The Lion, the Witch and the Wardrobe"
    Description: "Test your knowledge of C.S. Lewis's magical adventure"
  4. Add Questions Click the "+ Add Question" button to create a new question. For each question:
    • Enter the question text
    • Add 4 answer options (A, B, C, D)
    • Select which answer is correct
  5. Add More Questions Keep clicking "+ Add Question" until you have all your questions (typically 5-15 questions per quiz).
  6. Review and Save Double-check all questions and answers, then click "Save Quiz".
üí° Writing Good Questions:
  • Make questions clear and specific
  • Use age-appropriate vocabulary
  • Avoid trick questions or confusing wording
  • Make sure the correct answer is clearly correct
  • Make distractors (wrong answers) plausible but clearly incorrect
  • Mix question types: plot, character, theme, vocabulary

‚ú® Editing Existing Quizzes

You can modify any quiz at any time.

How to Edit

  1. Find the Quiz In the Admin Panel, scroll through the quiz list to find the one you want to edit.
  2. Click "Edit" Click the yellow "Edit" button on the quiz card.
  3. Make Changes The quiz will load in the manual editor where you can:
    • Change the quiz title or description
    • Edit any question text
    • Modify answer options
    • Change which answer is correct
    • Add new questions
    • Remove questions (click "Remove" on any question)
  4. Save Changes Click "Save Quiz" when done. Your changes will be immediately available to students.

üîÑ Hybrid Workflow (Best of Both Worlds)

Combine AI speed with manual precision for the best results.

Recommended Process

  1. Generate with AI Use AI generation to create a complete quiz quickly (8-10 questions recommended).
  2. Review Questions After AI generation, review each question in the editor that appears.
  3. Edit as Needed
    • Rephrase questions that are too complex or too simple
    • Adjust answer options to better match your students' level
    • Fix any questions that might be confusing
  4. Add Custom Questions Click "+ Add Question" to add your own questions on specific topics the AI might have missed.
  5. Remove Weak Questions Delete any AI-generated questions that don't meet your standards.
  6. Save the Final Quiz You now have a high-quality quiz that was quick to create but customized to your needs!
üí° Why This Works: AI saves you 90% of the time by generating good questions quickly. You spend 10% of the time polishing them to be perfect for your students. Best of both worlds!

Deleting Quizzes

‚ö†Ô∏è Warning: Deleting a quiz is permanent and cannot be undone. All student results for that quiz will still be preserved in the results database, but students won't be able to take the quiz again.

‚≠ê Best Practices

Creating Quality Quizzes

  • Start with 8-10 questions per quiz (not too long, not too short)
  • Mix easy, medium, and hard questions for variety
  • Cover different aspects: plot, characters, themes, vocabulary
  • Always review AI-generated questions before publishing
  • Test your quiz yourself to catch any errors
  • Update quizzes based on student feedback

Question Writing Tips

  • Keep questions focused on one concept
  • Use simple, clear language
  • Avoid negative questions ("Which is NOT...")
  • Make all answer options approximately the same length
  • Ensure only one answer is clearly correct
  • Don't use "All of the above" or "None of the above"

Difficulty Guidelines

Easy Questions about basic plot points, main characters, and obvious themes

Medium Questions requiring comprehension, character analysis, and understanding of relationships

Hard Questions about symbolism, deeper themes, author's purpose, and complex inferences

üîß Troubleshooting

AI Generation Issues

Problem: AI generation is taking too long (over 1 minute)

Solution: Refresh the page and try again. If it continues, switch to manual mode.

Problem: AI generates inappropriate or irrelevant questions

Solution: Use the "Additional Context" field to guide the AI more specifically. You can also edit or delete poor questions.

Saving Issues

Problem: Quiz won't save

Solution: Check that you've filled in all required fields (quiz title, question text, all 4 answer options, and selected a correct answer for each question).

Quiz Not Appearing

Problem: Saved quiz doesn't appear for students

Solution: Make sure the quiz ID follows the format subject-section-quizname. Check the curriculum structure in Firebase to ensure the subject and section exist.
